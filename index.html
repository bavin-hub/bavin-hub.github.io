General Loco-Manipulation

This project tackles one of robotics' hardest open problems: enabling a single robot to seamlessly combine locomotion and manipulation in unstructured, real-world environments, without relying on massive datasets or black-box end-to-end models.

The Problem

Existing approaches either require prohibitively expensive data collection, lack interpretability for safety-critical use, or fail to generalize beyond their training environment.

The Approach

A modular three-layer architecture decouples perception, neural reasoning, and symbolic control, making each component independently improvable and interpretable:

  • Perception layer - RGB-D camera, LiDAR, and IMU for 3D scene understanding and object pose estimation.
  • Neural layer - A Vision-Language Model (VLM) for open-vocabulary object grounding, paired with a Re-Identification (ReID) module that maintains object identity across occlusions.
  • Control layer - A PPO-trained reinforcement learning policy running on a Unitree G1 (29 DOF), trained in Isaac Lab simulation using a two-phase hierarchical reward that transitions the robot from navigation mode to manipulation mode at a 0.75m proximity threshold.
Entity - AI search engine with custom browser

RAG-Powered Search Engine | AI & Full-Stack

A search engine built around Retrieval-Augmented Generation, combining semantic search with a large language model to deliver grounded, contextually accurate answers, not just document links.

How the AI Works

Queries are converted into dense vector embeddings using HuggingFace's nomic-embed-text model and matched against a custom SQLite vector database built by crawling and indexing source URLs. The most relevant chunks are retrieved and passed as context to a Llama-instruct model on a HuggingFace inference endpoint, which synthesizes a coherent response anchored to the retrieved content, directly addressing hallucination and knowledge staleness common in vanilla LLMs.

Architecture

The React frontend is fully decoupled from the Flask backend via a REST API contract, making the AI pipeline swappable without touching the UI. The backend is modular by design: embedding model, vector store, and LLM can each be upgraded independently.

SimplRL - RL Algorithms from Scratch

Reinforcement Learning Algorithms | Research & Education

A from-scratch implementation of 9 core RL algorithms spanning value-based, policy-based, and actor-critic methods across both discrete and continuous action spaces, built directly from the original papers.

Covers the full modern RL stack: from foundational Q-Learning and DQN to state-of-the-art algorithms like PPO, SAC, and TD3. Each implementation is self-contained in a single file, heavily commented, and prioritizes clarity over abstraction, making it a practical reference for anyone studying or experimenting with RL concepts.

Algorithms

Q-Learning · DQN · Dueling Double DQN · REINFORCE · A2C · DDPG · PPO · SAC · TD3
